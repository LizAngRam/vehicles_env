## vehicles_env
Proyecto de análisis exploratorio de datos de vehículos usando Python. Incluye visualizaciones interactivas con Plotly Express y una aplicación web desarrollada con Streamlit para explorar el conjunto de datos de forma dinámica.
(Proyecto Sprint 7)
## Funcionalidad
La aplicación ofrece las siguientes funciones:
- Visualización de los datos en forma de tabla.
- Construcción de histogramas para columnas numéricas seleccionadas.
- Creación de gráficos de dispersión para analizar la relación entre distintas variables numéricas, como el precio y el kilometraje.

Los gráficos son interactivos y permiten al usuario seleccionar las variables que desea analizar mediante casillas de verificación y menús desplegables.

## Tecnologías utilizadas
- Python
- Pandas
- Plotly Express
- Streamlit

## Cómo ejecutar la aplicación
1. Instalar las dependencias listadas en `requirements.txt`.
2. Ejecutar el siguiente comando en la terminal:

```bash
streamlit run app.py
